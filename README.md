## üöÄ How to Use

### Method 1: Browser Console (Quickest)
1. Go to your [Roblox Profile](https://www.roblox.com/users/profile).
2. Right-click anywhere > **Inspect** (or press `F12`).
3. Click the **Console** tab.
4. *Note: If asked, type `allow pasting` and hit Enter.*
5. Copy the code from [Roblox-Mass-Unfollower.js](Roblox-Mass-Unfollower.js).
6. Paste it into the console and press **Enter**.
